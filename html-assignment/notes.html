HTML Tags Learned in MTM6130

HTML Tag < cite >
Identifies the title of a creative work or a citation, ex: A Tale of Two Cities

HTML Tag < sub >
Defines subscript text, ex: H20

HTML Tag < abbr >
Defines an abbreviation or and acronym, ex: HTML

HTML Tag < strong >
Defines highly emphasized/important text, ex: Strong

HTML Tag < q >
Defines a small quotation that is embedded within other content, ex: live, laugh, love
